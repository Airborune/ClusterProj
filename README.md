# Streaming Time Series Clustering

Учебный проект для потоковой кластеризации временных рядов на примере ECG200.

Реализованы два метода:

- `CluStream`: микрокластеры с ограничением их количества.
- `DenStream`: плотностная потоковая кластеризация с потенциальными кластерами, outlier-кластерами и шумом.
